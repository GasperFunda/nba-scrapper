# NBA Scrapper
A web scrapper written in Java for extracting the 3 point average for a season given the player name using the Jsoup library.

# Usage:
INPUT: Lebron James
OUTPUT: 
2003-04 0.8
2004-05 1.4
2005-06 1.6

2007-08 1.5
2008-09 1.6
2009-10 1.7
2010-11 1.2
2011-12 0.9
2012-13 1.4
2013-14 1.5
2014-15 1.7
2015-16 1.1
2016-17 1.7
2017-18 1.8
2018-19 2.0
2019-20 2.2
