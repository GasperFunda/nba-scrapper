# NBA Scrapper
A web scrapper written in Java for extracting the 3 point average for a season given the player name using the Jsoup library.

# Usage:
INPUT: Lebron James <br>
OUTPUT:  <br>
2003-04 0.8 <br>
2004-05 1.4 <br>
2005-06 1.6 <br>
2006-07 1.3 <br>
2007-08 1.5 <br>
2008-09 1.6 <br>
2009-10 1.7 <br>
2010-11 1.2 <br>
2011-12 0.9 <br>
2012-13 1.4 <br>
2013-14 1.5 <br>
2014-15 1.7 <br>
2015-16 1.1 <br>
2016-17 1.7 <br>
2017-18 1.8 <br>
2018-19 2.0 <br> 
2019-20 2.2 <br>
2020-21 2.3 <br>
<br>
IDE: IntelliJ <br>
Built with: Maven
